📝Publier un article

Chaque article de blog (Insights juridiques) est une page HTML séparée, reliée depuis la section "Insights" de la page d'accueil. Voici comment procéder sans aucune connaissance en code.

Méthode recommandée — GitHub + Netlify (gratuit)

Créer un compte GitHub

Rendez-vous sur github.com → créez un compte gratuit. GitHub est votre "gestionnaire de fichiers en ligne" pour votre site.

Uploader vos fichiers HTML dans un dépôt

Créez un nouveau dépôt (repository) nommé rootseedlegal-site. Glissez-déposez tous vos fichiers HTML dans ce dépôt.

Créer votre article

Dupliquez le fichier article-modele.html (fourni ci-dessous), renommez-le (ex: article-rgpd-2025.html), et remplissez le titre, le texte, la date et la catégorie.

Ajouter le lien dans index.html

Dans index.html, trouvez la section <!-- BLOG / INSIGHTS --> et ajoutez une carte d'article en copiant le bloc .blog-card existant, en modifiant le titre, le résumé et le lien href.

Déployer sur Netlify

Connectez votre dépôt GitHub à Netlify (paramètres → "Link to Git"). À chaque modification sur GitHub, le site se met à jour automatiquement en 30 secondes.

💡 Astuce : Si vous ne voulez pas toucher au code HTML, utilisez Netlify CMS (gratuit) qui s'ajoute à votre site et vous permet d'écrire vos articles comme dans Word, via une interface graphique. Je peux vous configurer ça sur demande.

📄Modèle HTML d'article

Copiez ce modèle, enregistrez sous un nouveau nom (ex: article-contrats-saas.html) et remplissez les zones en MAJUSCULES :

<!-- CARTE À AJOUTER DANS index.html (section Blog) -->
<div class="blog-card fade-in">
  <div class="blog-img">⚖️</div>  <!-- Changer l'emoji -->
  <div class="blog-body">
    <span class="blog-category">CATÉGORIE (ex: Droit des Affaires)</span>
    <h3 class="blog-title">TITRE DE L'ARTICLE</h3>
    <p class="blog-excerpt">RÉSUMÉ EN 1-2 PHRASES.</p>
    <a href="articles/NOM-DU-FICHIER.html" class="blog-link">Lire l'article</a>
  </div>
</div>
⚠️ SEO : Pour chaque article, personnalisez les balises <title>, <meta name="description"> et <link rel="canonical">. C'est ce qui permet à Google d'indexer et positionner chaque article séparément.

🎙️Ajouter un lien d'interview

Pour ajouter une interview (podcast, article de presse, YouTube, radio), deux options :

Option A — Ajouter dans la section Blog (rapide)

Traitez l'interview comme un article. Dans la section Blog d'index.html, ajoutez une carte avec :

<div class="blog-card fade-in">
  <div class="blog-img">🎙️</div>
  <div class="blog-body">
    <span class="blog-category">Interview · NOM DU MÉDIA</span>
    <h3 class="blog-title">TITRE DE L'INTERVIEW</h3>
    <p class="blog-excerpt">Me Belkacem interviewée sur [SUJET] par [MÉDIA]. DATE.</p>
    <a href="URL_DIRECTE_DE_L_INTERVIEW"
       target="_blank" rel="noopener" class="blog-link">
       Écouter / Voir l'interview
    &a>
  </div>
</div>

Option B — Créer une page "Presse & Médias" dédiée (recommandé)

Une page dédiée est plus professionnelle et améliore le SEO sur vos requêtes de notoriété (recherche de votre nom). Elle peut contenir :

  • Logos des médias qui vous ont cité
  • Liens vers chaque interview/podcast/article
  • Date et sujet de chaque intervention
  • Extraits ou citations clés
💡 Demandez-moi de générer cette page "Presse & Médias" en envoyant la liste de vos interviews avec : titre, média, date, URL, et je crée la page complète prête à l'emploi.

🚀Déployer et gérer sur Netlify

Créer un compte Netlify

Rendez-vous sur netlify.com → Sign up → choisissez "Email" ou connectez avec GitHub.

Déploiement par glisser-déposer

Dashboard → "Add new site" → "Deploy manually" → glissez-déposez le dossier rootseedlegal (le dossier dézippé, pas le ZIP). Le site est en ligne en 30 secondes avec une URL du type amazing-site-123.netlify.app.

Ajouter votre domaine rootseedlegal.com

Site settings → Domain management → "Add custom domain" → entrez rootseedlegal.com → suivez les instructions pour pointer votre DNS vers Netlify (modifier les nameservers chez votre registrar).

HTTPS automatique

Netlify active automatiquement Let's Encrypt SSL. Votre site sera accessible en https://rootseedlegal.com — essentiel pour le SEO et la confiance des visiteurs.

Modifier un texte rapidement (méthode simple)

  1. Ouvrez le fichier HTML avec Bloc-notes (Windows) ou TextEdit (Mac) — ou mieux avec VS Code (gratuit)
  2. Utilisez Ctrl+F pour trouver le texte à modifier
  3. Modifiez le texte entre les balises HTML (ne touchez pas aux balises elles-mêmes)
  4. Sauvegardez et re-déployez sur Netlify

🔍Google Search Console & Analytics

Google Search Console (gratuit — indispensable)

Allez sur search.google.com/search-console → "Ajouter une propriété" → entrez rootseedlegal.com → vérifiez la propriété (Netlify permet la vérification par fichier HTML) → soumettez votre sitemap.xml.

Google Analytics 4 (gratuit — recommandé)

Allez sur analytics.google.com → créez une propriété → copiez le code de suivi (une ligne de JavaScript) → collez-le dans le <head> de chaque page HTML. Vous verrez ensuite combien de visiteurs arrivent, depuis quelle source, sur quelle page.

✅ À faire dans les 48h après mise en ligne : (1) Soumettre le sitemap dans Search Console → Google indexe le site en 3-7 jours. (2) Demander l'indexation manuelle page par page pour accélérer.

📍Google My Business (SEO Local)

La fiche Google My Business est l'action SEO local la plus importante pour un cabinet d'avocats parisien. Elle vous positionne dans le Google Maps Pack (les 3 résultats qui apparaissent sur les cartes) et dans les résultats locaux.

Créer/revendiquer votre fiche

Rendez-vous sur business.google.com → "Gérer maintenant" → recherchez "Root & Seed Legal" → si la fiche existe, revendiquez-la ; sinon créez-la.

Remplir 100% du profil

Catégorie : "Cabinet d'avocats" | Adresse : 9 rue Victorien Sardou, 75016 Paris | Téléphone | Site : rootseedlegal.com | Horaires | Description (250 mots avec mots-clés)

Ajouter des photos

Photo de profil (logo), photos du cabinet, photo de Me Belkacem (optionnel). Les fiches avec photos reçoivent 42% de demandes d'itinéraire et 35% de clics supplémentaires.

Obtenir des avis Google

Demandez à vos clients satisfaits de laisser un avis. Un cabinet avec 10+ avis positifs est favorisé par Google. Répondez toujours aux avis (positifs et négatifs).
