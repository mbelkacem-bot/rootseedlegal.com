# Root & Seed Legal — Site web

## Déploiement

### Via Netlify (drag & drop)
1. Aller sur https://app.netlify.com
2. Onglet "Sites" → "Add new site" → "Deploy manually"
3. Glisser-déposer CE DOSSIER entier sur la zone de dépôt
4. Le site est en ligne en 30 secondes ✓

### Via Netlify + GitHub
Voir les instructions de déploiement ci-jointes.

## Fichiers
- `index.html` — Site complet (tout-en-un, inclut CSS, JS et images en base64)
- `netlify.toml` — Configuration Netlify (en-têtes sécurité, redirections)
- `_redirects` — Règles de redirection Netlify (fallback)
