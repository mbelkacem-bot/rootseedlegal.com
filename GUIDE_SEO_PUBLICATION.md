## ÉTAPE 1 — ACTIVER FORMSPREE (10 minutes)

### Pourquoi Formspree ?
Formspree intercepte les soumissions de vos formulaires et vous envoie un email à chaque fois qu'un prospect remplit le formulaire. Gratuit jusqu'à 50 soumissions/mois.

### Instructions :

1. Allez sur **https://formspree.io**
2. Créez un compte gratuit avec votre email : secretariat@rootseedlegal.fr
3. Cliquez sur **"+ New Form"**
4. Donnez-lui un nom : "Contact Root & Seed Legal"
5. Copiez l'**ID unique** que Formspree vous donne (format : `xpwzjkle`)
6. Dans vos fichiers HTML, remplacez partout :
   ```
   https://formspree.io/f/VOTRE_ID_FORMSPREE
   ```
   par :
   ```
   https://formspree.io/f/xpwzjkle
   ```
   (remplacez `xpwzjkle` par votre vrai ID)

### Ce qui se passe ensuite :
✅ Un prospect remplit le formulaire
✅ Formspree reçoit les données
✅ **Vous recevez un email sur secretariat@rootseedlegal.fr** avec :
   - Nom & Prénom
   - Email
   - Téléphone
   - Domaine concerné
   - Message

## ÉTAPE 2 — PUBLIER LE SITE SUR NETLIFY (5 minutes, gratuit)

### Pourquoi Netlify ?
- 100% gratuit pour votre usage
- Déploiement en glisser-déposer
- HTTPS automatique
- CDN mondial (chargement rapide)
- Compatible avec votre domaine rootseedlegal.com

### Instructions :

1. Allez sur **https://www.netlify.com**
2. Créez un compte gratuit
3. Sur votre tableau de bord, faites glisser votre **dossier complet** (contenant index.html, uhnwi.html, etc.) dans la zone "Déposer ici"
4. Netlify génère une URL temporaire (ex: amazing-einstein-123.netlify.app)
5. Dans **Settings > Domain Management > Add custom domain**, entrez : rootseedlegal.com
6. Netlify vous donne les DNS à configurer chez votre registrar
7. **Votre site est en ligne en moins de 5 minutes** à www.rootseedlegal.com
